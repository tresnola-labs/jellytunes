# Tests BDD para JellyTunes

Este directorio contiene la suite de tests BDD (Behavior-Driven Development) para JellyTunes.

## Estructura

- `features/` - Escenarios Gherkin (.feature)
- `steps/` - Implementación de steps en TypeScript
- `support/` - Hooks, world y utilidades
